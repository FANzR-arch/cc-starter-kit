# Writing Skill 示例

> 这是一个写作类 Skill 的示例，展示 Skill 的基本结构。
> 复制这个文件夹，改成你自己的工作流。

## 触发条件

当用户说以下关键词时激活：
- 帮我写文章
- 写一篇
- 起草

## 执行步骤

1. 先问清楚：主题是什么、目标读者是谁、大概多长
2. 给出一个简短的结构草稿，确认方向
3. 用户确认后再写完整版
4. 写完问：需要调整语气或结构吗？

## 输出格式

- 用 Markdown 格式输出
- 标题用 H1，段落之间空一行
- 结尾附上一句"如需调整，告诉我哪里不对"

## 参考素材

如果有写作风格参考或模板，放在 `references/` 文件夹里，
并在这里注明：`Read: references/风格参考.md`
